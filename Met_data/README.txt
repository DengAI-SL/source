stations.csv: Contains the GPS coordinates of weather stations (rainfall) which are having data for the whole year. 

allStations.csv: Contains weather stations (rainfall) after filling the missing values. The final 24 stations are the stations which are missing less than 2 months of data.

Temperature.csv: Contains GPS coordinates of weather stations (temperature). 

moh_temp.csv: Intersection of MOH areas and temperature stations. This file is used to find the temperature stations which are corresponding to each MOH area.

moh_weather.csv: Intersection of MOH areas and weather (rainfall) stations.

corrected_weather: contains data of all the rainfall data used for this project. 

\corrected_weather\mohWeather.csv: weekly rainfall on MOH basis.

Removed: rainfall data of weather stations which do not contain data for the whole year.
